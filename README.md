# OrangeHRM Web Automation

Automated end-to-end test suite for the OrangeHRM demo application  
(https://eaapp.somee.com) using Selenium WebDriver and Java.

## Modules Automated
- Login
- Add Employee
- Logout

## Tech Stack
Java| Selenium WebDriver |TestNG | Page Object Model | Maven

## How to Run
1. Clone the repo
2. Open in Eclipse / IntelliJ
3. Run `testng.xml` as TestNG Suite

## Key Concepts Covered
- @Test, @BeforeMethod, @AfterMethod annotations
- Assertion-based validations
- Reusable page actions via POM
- Suite execution via testng.xml
